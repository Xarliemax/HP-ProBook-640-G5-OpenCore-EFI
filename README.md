An OpenCore EFI for HP ProBook 640 G5 Laptops, you can copy this to a Hackintosh to get all fuctionalities.

What MacOS version?:

It works Up to Tahoe 26 (Latest), older versions are not tested, but i recommend Sonoma (14.4) and later.

Fuctionalities:

- Wi-Fi (Native OK)
- Bluetooth (OK)
- Audio (OK)
- USB (OK)
- Internal Speakers (OK)
- Battery State (OK)
- Hardware Acceleration (OK)
- Plug and Play (OK)
- Startup Chime (Not Working)
- iPhone Duplication (Imposible on Hackintosh)
- Airdrop (Not Working)

IMPORTANT INFO:

Before installing and configuring MacOS, you need to download OCLP-Mod and install the Root Patches, do this or Wi-Fi and 
internal speakers audio won't work.
